# docs/ — Public documentation

This folder is in the **public** repo. Do not put SEA-specific or private information here:

- No real server IPs, hostnames, or domain names
- No BetterStack monitor names, heartbeat names, or URLs
- No GitHub secret names or values
- No SSH host aliases or usernames
- No org-specific configuration details

Infrastructure-specific docs (monitoring setup, server details, deployment specifics) go in **`private/docs/`** instead.

Docs here should be generic enough to work for any deployment of this project.
